Chronologie des Langages

1843

Premier algorithme par Ada Lovelace

Ada Lovelace écrit le premier algorithme destiné à être traité par une machine, la machine analytique de Charles Babbage, faisant d'elle la première programmeuse de l'histoire.

// Premier algorithme - concept du moteur analytique
N₀ = 0
N₁ = 1
Pour chaque n > 1:
Nₙ = Nₙ₋₁ + Nₙ₋₂

1957

FORTRAN - Premier langage de haut niveau

Développé par IBM, FORTRAN (FORmula TRANslation) devient le premier langage de programmation de haut niveau largement utilisé, révolutionnant la façon dont les scientifiques et ingénieurs interagissent avec les ordinateurs.

PROGRAM HELLO
PRINT *, 'Hello, World!'
END PROGRAM HELLO

1972

Le langage C

Créé par Dennis Ritchie aux Bell Labs, C devient un langage puissant qui influence de nombreux langages modernes et permet le développement d'UNIX, posant les bases de l'informatique moderne.

#include <stdio.h>
int main() {
  printf("Hello, World!\n");
  return 0;
}

1989

Naissance du World Wide Web et HTML

Tim Berners-Lee invente le World Wide Web au CERN, avec HTML comme langage de balisage pour créer les premières pages web, transformant à jamais notre façon d'accéder à l'information.

<!DOCTYPE html>
<html>
  <head>
    <title>Ma première page</title>
  </head>
  <body>
    <h1>Hello, World!</h1>
  </body>
</html>
